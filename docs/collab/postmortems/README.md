# 误判 / 事故复盘（Postmortems）

每起严重问题一篇，文件名：`YYYY-MM-DD-short-title.md`。

## 模板

```markdown
# 标题

## 摘要
一句话：发生了什么、影响是什么。

## 影响
- 错误推荐 / 错误 NO-GO / 状态污染 / 成本事故 / 隐私泄漏 …

## 时间线
- HH:MM …

## 根因
- 直接原因
- 深层原因（Prompt / 缺证据 / 状态未读 / 限额失效 …）

## 修复
已做 / 待做

## 预防
测试、验收项、Prompt 版本、Runbook 条目
```
